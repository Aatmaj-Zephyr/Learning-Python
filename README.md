2) Loops and nested loops

3) Lists, tuples and dictionaries

4) String functions

B) Intermediate-

1) User defined functions

2) GUI and design

3) File handling in Python

4) OOP techniques

c) Advanced-

1) MySql

2) Database connectivity

3) Form designing

4) Interesting Projects
